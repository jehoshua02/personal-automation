# Plugin Extraction

## Abstract

Evaluate which agent instructions in CLAUDE.md and global rules can be lifted into reusable private plugins. Scope: project-level and user-level instructions not already coming from a plugin. Reduce per-project boilerplate.

## Priority: 55

- Value: 5/10 — Reduces boilerplate across projects, but only benefits user.
- Momentum: 1/10 — Greenfield.
- Effort: 5/10 — Audit instructions, design plugin boundaries, create and test.
- Risk: 2/10 — Low stakes, plugins can be iterated.
